# Dynamically change UITableView Footer or Header TextLabel

```
tableView.headerView(forSection: 0)?.textLabel?.text = "\(dataSource[row])"
tableView.footerView(forSection: 0)?.textLabel?.text = "\(dataSource[row])"
```
